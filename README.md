# Password Strength Analyser

## Overview
C++ application for the Data Structures and Algorithms course at T.A. Marryshow Community College. Analyses password security using hash tables and trie data structures.

## Features
- Password strength scoring
- Common password detection
- Secure password generation
- Hash table and trie implementations

## Project Files
- `PasswordStrengthAnalyser.cpp` - Main application
- `pseudocode.txt` - Algorithm design

## Technology
- **C++** - Programming language
- **Hash Tables** - Password storage
- **Tries** - Pattern matching

## Educational Purpose
This project is for educational and demonstration purposes showcasing data structures and algorithm implementation in C++.
